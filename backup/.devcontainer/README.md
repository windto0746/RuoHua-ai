# Dev Container 配置

此配置用于 GitHub Codespaces，让用户可以在浏览器中直接运行项目。

## 使用方法

1. 在 GitHub 仓库页面，点击 **Code** → **Codespaces** → **New codespace**
2. 等待环境创建完成
3. 在终端中运行：`python app.py`
4. 点击 "Open in Browser" 访问应用

## 环境要求

- Python 3.10+
- pip install gradio dashscope

## 注意事项

- 需要配置 DashScope API Key 才能正常使用AI功能
- API Key 可在阿里云DashScope官网申请
